# ahuth.github.io

Personal homepage.

## Prerequisites

1. [Node 6.11+](https://nodejs.org/en/)
2. [Yarn 1.5+](https://yarnpkg.com/)

## Installation

Run `yarn install` to install all Javascript dependencies.

## Usage

To run in development mode, run:

```
yarn run dev
```

and then navigate to:

```
localhost:3000
```
